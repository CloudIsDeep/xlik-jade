#### 目标种类

```
中立：指的是中立无敌意（中立被动），是一个玩家。所以中立敌对不属于此类。
古树：单位-状态-单位类别-古树，如生命之树、战争古树
机械：单位-状态-单位类别-机械类，如坦克、粉碎者以及非古树的建筑
有机：非机械类单位，例如农民、步兵，石傀儡也是有机单位
残骸：可破坏物-战斗-作为目标类型-残骸，如大门、木桶
装饰品：很明显是地形装饰物，会使装饰品单位无法被攻击和不受伤害，但会被主要类型如“地面”之类的优先，所以不能与“地面”之类同时选中，否则无效【这段我没看懂，所以没修改。。】
没有：和装饰品差不多，不过更加全面，即使选了主要类型如“地面”之类的再选择没有，这个单位就相当于无敌，会优先于其它所有选项 【也就是说，一个技能只要勾了这一项，那么它对任何东西都无效】
地形：性质和装饰物相同
友军单位：指自己的部队和盟友的部队
别人：除该单位以外的任何单位
联盟：就单指盟友的部队,不包括自己的部队
自己：该单位
守卫：单位-状态-单位类别-守卫，如毒蛇守卫、治疗守卫
墙：可破坏物-战斗-作为目标类型-墙，如石头之墙
玩家单位：该单位所有者的单位，不包括其他玩家的单位
无敌的：无敌的单位
可攻击的：可以被攻击的单位，也就是非-无敌的
空中：单位-战斗-作为目标类型-空中。移动类型不一定是飞行的，不过穿刺技能则不能应用于移动类型是飞行和目标类型是空中的单位
地面：单位-战斗-作为目标类型-地面
英雄：不用说都知道
自爆工兵：单位-状态-单位类别-自爆工兵
存活、死亡：活着的单位、死亡的单位
建筑：单位-状态-是一个建筑-TRUE
```

#### 目标分类

从A到H，8组。组内是兼容的，即满足其中一个就可执行，组间是不兼容的，即某个物体要成为目标允许的对象，而且已经满足了其中一组的条件，那么它必须也要满足其它7组的条件。

```
<*> 空中、地面、建筑、守卫是4大类。这4项必须至少选一项，否则单位就无法战斗。默认不选。
<A组> 中立、敌人、友军，默认为全选（玩家单位、结盟的，其实这两项应该是朋友的支 项，也可以将A组分成中立、敌人、玩家单位、结盟的）。
<B组> 别人、自己，默认为全选。
<C组> 古树、非古树的，默认为全选。
<D组> 可攻击的、无敌的，默认为可攻击的。
<E组> 有机的、机械的，默认为全选。
<F组> 死亡、存活，默认为活的。
<G组> 英雄、非英雄，默认为全选。
<H组> 自爆工兵、非自爆工兵，默认全选。
```

注：默认是指这一组内你任意选型一个都不勾选
